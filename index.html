Facial Recognition App

An app where users are able to register an account and submit a photo url for single face detection. Users can sign in and out and may also delete their accounts. The application also keeps track of the amount of submissions the user has done.

Technology Used

  • PostgreSQL

Robot Contacts

This is a front-end application that displays card components for each user stored in the JSON placeholder api. The user name and email data is rendered inside of its own card component and is displayed within a searchable scroll area.

Technology Used
  • React
  • Redux
  • Heroku

JavaScript Drumkit

This site was built using vanilla javascript. Users are able to play drum sounds by either clicking the on-screen buttons or entering the corresponding keys. Vanilla JavaScript was used to write this page, no libraries such as jQuery were utilized.

Technology Used
